# FlashBack — 프로젝트 설명 (200단어)

**FlashBack**은 NVIDIA Cosmos-Reason2를 활용하여 화재 안전의 핵심 과제인 **감시 영상에서 화재 발원지를 역추적**하는 시스템입니다. 기존 시스템이 화재 존재 여부만 감지하는 것과 달리, FlashBack은 시간에 따른 화재 *확산 과정*을 분석하여 *어디서 시작되었는지*를 추론합니다.

감시 카메라 프레임 시퀀스를 시간적 영상으로 변환한 뒤, Cosmos-Reason2의 물리 추론 능력으로 화염 전파 패턴, 연기 분산 역학, 대류 기반 확산을 분석합니다. 이 패턴을 시간 역순으로 추적하여 발원지를 텍스트 설명("온실 좌측 하단")과 정밀한 이미지 좌표(x=0.25, y=0.75) 모두로 추정합니다.

**핵심 혁신**: 발원지 좌표를 원본 프레임 위에 직접 시각화합니다 — 빨간 십자선이 예측 발원지를, 노란 화살표가 확산 방향을 표시합니다. Lucas-Kanade 광학 흐름으로 카메라 움직임에 따라 발원지를 프레임마다 추적하여 데모 영상으로 제작합니다.

**결과**: 11개 화재 장면(FLAME/SMOKE/NORMAL)에서 발원지 추론율 100%, 시간적 추론율 100%를 달성했습니다.

**물리 추론의 실제**: 모델은 대류 패턴, 연료 밀도 상관관계, 시간적 확산 특성을 통해 발화점을 설명하며, Chain-of-Thought로 분석 과정을 단계별로 제시합니다.
